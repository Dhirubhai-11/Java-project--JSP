## Add Two Numbers Using JSP

This is a simple web application developed using JSP in NetBeans.
It allows users to enter two numbers and displays their sum.

## Technologies Used

Java
JSP
HTML
Apache Tomcat Server
NetBeans IDE

## How to Run

Clone the repository
Open in NetBeans
Run the project using Tomcat server
Open in browser:
http://localhost:8080/ProjectName

## How It Works

User enters two numbers.
JSP retrieves the values using request.getParameter().
The numbers are added and the result is displayed.
